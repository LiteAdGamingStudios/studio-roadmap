# LiteAds Gaming Studios

## Vision

LiteAds Gaming Studios is a small independent mobile game studio focused on creating lightweight, fun-first games with minimal advertisements and clean user experiences.

We believe mobile gaming should be enjoyable without overwhelming players with aggressive ads, forced monetization, or unnecessary complexity.

Our goal is to build games that are:

* Simple and fun
* Lightweight and accessible
* Respectful of player time
* Minimal in advertisements
* Built with passion and experimentation

---

## Our Approach

We are starting small and growing step by step.

### Phase 1

Create a mini-game hub containing multiple small mobile games inside a single application.

### Phase 2

Develop standalone casual mobile games with improved gameplay and polish.

### Phase 3

Move toward more advanced and intermediate-level games with better systems, progression, and long-term support.
